# Dotnet Core VueJs template

## Prerequisites
 - Node v12.14.1 +
 - DotNet Core 3.1

## Running in dev

### Start Client App
The static files in dev are served by the webpack dev server
 - cd ClientApp
 - npm install
 - npm run serve:ssr

 ### Start dotnet server
 - dotnet run

 ## Publishing
 - dotnet publish
